Other Component Upgrades

Delete Confirmation

shadcn AlertDialog with red "Delete" button. Shows what will be deleted. "This action cannot be undone."

Toast Notifications

shadcn Sonner toasts. Success (green check), Error (red x), Info (blue). Bottom-right position. Auto-dismiss after 4s.

Loading → Skeletons

shadcn Skeleton components that mirror actual layout. Cards become gray pulsing rectangles. Tables show row placeholders.

Empty States

Lucide icon (muted, 48px) + heading + description + CTA button. No emoji. Clean, actionable, professional.

Animations

Sheets slide in 200ms ease-out. Dropdowns fade+scale 150ms. Toasts slide up. Skeleton → content crossfade. Page transitions via top progress bar (nprogress).

Action Menus

Replace inline "Editar" / "Eliminar" text with a ••• button that opens a shadcn DropdownMenu with Edit, Delete, and contextual actions.
