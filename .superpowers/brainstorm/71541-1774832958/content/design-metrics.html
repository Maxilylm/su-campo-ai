Design Section 5: Métricas (KPIs & Dashboard)

Snapshot on top, trends below — the farm's control panel

Architecture: No New Tables

Métricas is a read-only aggregation layer. It queries existing tables — no new data model needed. All KPIs are computed from: cattle, crops, crop_applications, inventory_items, inventory_movements, financial_transactions, vaccinations, health_events.

AI Chat Examples

🎙️ "¿Cómo va el campo?"
→ Returns snapshot: total heads, planted ha, overdue items, financial summary
🎙️ "¿Cuál es mi carga animal?"
→ Calculates total heads / total hectares
🎙️ "¿Qué cultivo me rindió mejor?"
→ Compares yield_per_hectare across harvested crops
🎙️ "¿Cuánto gasté en sanidad este año?"
→ SUM financial_transactions WHERE category IN (veterinario, medicamento) AND year

📊 Charting Library

Use Recharts (built on D3, React-native, lightweight). Already compatible with Next.js + React 19. Bar charts for monthly trends, line charts for time series. No heavy dashboarding framework needed — the KPIs are simple aggregations rendered as cards + charts.
