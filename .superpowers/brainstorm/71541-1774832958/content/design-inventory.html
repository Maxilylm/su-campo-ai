Design Section 3: Inventario (Consumable Inputs)

Track what the farm uses — feed, seeds, fertilizer, medicine, fuel

Data Model

CREATE TABLE inventory_items (
  id UUID PRIMARY KEY,
  farm_id UUID REFERENCES farms ON DELETE CASCADE,
  name TEXT NOT NULL, -- "Ración Bovina 18%", "Glifosato 48%", etc.
  category TEXT NOT NULL, -- alimento, semilla, fertilizante, agroquímico, medicamento, combustible, otro
  unit TEXT NOT NULL, -- kg, L, dosis, unidad
  current_stock NUMERIC NOT NULL DEFAULT 0,
  min_stock NUMERIC, -- alert threshold
  cost_per_unit NUMERIC, -- latest purchase price per unit (feeds into Finanzas)
  notes TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE inventory_movements (
  id UUID PRIMARY KEY,
  farm_id UUID REFERENCES farms ON DELETE CASCADE,
  item_id UUID REFERENCES inventory_items ON DELETE CASCADE,
  type TEXT NOT NULL, -- compra, uso, ajuste, pérdida
  quantity NUMERIC NOT NULL, -- positive for compra, negative for uso
  unit_cost NUMERIC, -- price paid per unit (compra only)
  total_cost NUMERIC GENERATED ALWAYS AS (quantity * COALESCE(unit_cost, 0)) STORED,
  section_id UUID REFERENCES sections, -- where it was used (for cost allocation)
  crop_id UUID REFERENCES crops, -- which crop consumed it
  cattle_id UUID REFERENCES cattle, -- which herd consumed it
  date DATE NOT NULL DEFAULT CURRENT_DATE,
  notes TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- current_stock is updated via trigger on inventory_movements insert
-- cost_per_unit is updated to latest compra's unit_cost via trigger

How Inventory Connects to Other Sections

→ Agricultura When a crop_application is logged, it can auto-deduct from inventory (fertilizer, herbicide used) → Sanidad When a vaccination is recorded, it can auto-deduct medicine doses from inventory → Finanzas Every "compra" movement is an expense. "uso" movements allocate costs to sections/crops/herds for cost-per-unit calculation → Métricas Consumption trends, cost trends, restock frequency — all derived from inventory_movements → Chat AI "Compré 1000kg de ración a $280 el kilo" → creates item (if new) + compra movement

AI Chat Examples

🎙️ "Compré 1000 kilos de ración a 280 pesos el kilo"
→ Creates/updates inventory item + compra movement (stock +1000, cost $280/kg)
🎙️ "Usé 50 litros de gasoil en el potrero Norte"
→ Creates uso movement (stock -50), allocated to section Norte
🎙️ "¿Cuánta ración me queda?"
→ Query: returns current_stock for matching item
🎙️ "¿Qué me falta comprar?"
→ Query: returns all items where current_stock < min_stock
