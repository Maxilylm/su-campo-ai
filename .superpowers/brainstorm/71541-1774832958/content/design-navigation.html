Shared State: Farm Context Provider

The root layout.tsx wraps everything in a FarmProvider context that holds:

  • farm — id, name, operation_type, total_hectares
  • sections — all sections (shared across Hacienda, Agricultura, Mapa, Inventario)
  • refreshFarm() — refetch after mutations

Each page fetches its own domain data (cattle, crops, transactions, etc.) independently. Only the farm + sections are shared — because almost every page needs them.
